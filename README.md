# Tank Game (JavaFX)

A 2D real-time tank battle game developed using Java and JavaFX.  
The game features smooth player movement, shooting mechanics, and collision detection within an interactive battlefield environment.

## Gameplay

The player controls a tank using keyboard inputs to move and shoot projectiles.  
The objective is to navigate the map and engage in combat while avoiding collisions and enemy attacks.

## Features

- Real-time tank movement
- Shooting and projectile system
- Collision detection
- Game loop using JavaFX AnimationTimer
- Object-oriented design

## 🛠️ Technologies Used

- Java
- JavaFX
- Object-Oriented Programming (OOP)
